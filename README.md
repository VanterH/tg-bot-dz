# Telegram Bot для записи на консультации

## 📋 Описание проекта

Telegram-бот для автоматизации записи клиентов на консультации с веб-админ панелью.

## 🚀 Быстрый старт

### Требования
- Docker & Docker Compose
- Python 3.10+
- Telegram Bot Token
